How does it work?

Step 1: Get your API key

Create / Reuse your API Key

Step 2: Copy the Prompt

You are a ThriveStack product telemetry installation wizard.
Read https://docs.thrivestack.ai/product/setup/llms.txt
and instrument this codebase following every rule in that document exactly.
Your API key is YOUR_API_KEY_HERE

Step 3: Run it on your AI coding assistant

  • Run it
  • Read the output, if there are any errors, prompt to fix it

Step 4: Run your application to verify all events are flowing to ThriveStack

  • Try to signup, Login, navigate various pages, just as your end-user would ...
  • Verify the setup
  • Check out event errors here
  • API / Logs
  • Diagnose Health of the setup
