Setup with AI Coding Assistants (Backend)

Use an AI coding assistant to instrument your backend or API server with ThriveStack product telemetry via HTTP — no browser SDK required.

and more
Node.js Python Go / Ruby / Java
This guide is for backend / server-side codebases only — Node.js API servers, Python (Django / FastAPI / Flask), Go, Ruby on Rails, Java, etc. If your codebase runs in a browser, use the frontend setup guide instead.

Step 3: Run it on your AI coding assistant

  • Open your backend repo in your AI coding assistant
  • Paste the prompt and run it
  • The AI will ask you a few questions (B2B vs B2C, feature types) — answer them before it writes any code
  • If there are errors in the output, ask the AI to fix them

Step 4: Verify events are flowing to ThriveStack

  • Trigger a signup or login in your app
  • Check your server logs for outbound POST calls to api.app.thrivestack.ai
  • Check out event errors here
  • API / Logs
  • Diagnose Health of the setup
Note: Unlike the frontend guide, the backend guide sends events via direct HTTP calls — no script tag is installed. The AI creates a small reusable helper function and calls it from each instrumented handler.
